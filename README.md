# 🎉 나의 첫 번째 웹 페이지

환영합니다! 이것은 HTML로 만든 첫 번째 웹 페이지입니다.

## 📋 프로젝트 설명

이 프로젝트는 아름답고 현대적인 디자인의 단일 HTML 페이지입니다.

### ✨ 주요 기능

- 🎨 아름다운 그라데이션 배경
- 💫 부드러운 애니메이션 효과
- 🔘 인터랙티브 버튼
- 📱 반응형 디자인 (모바일, 태블릿, PC 모두 지원)

## 🚀 사용 방법

1. `index.html` 파일을 더블클릭하거나
2. 브라우저에서 파일을 열면 됩니다!

## 🛠️ 기술 스택

- HTML5
- CSS3
- JavaScript
